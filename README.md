# WB расчеты по детализации

Мини-веб-приложение для расчета показателей по еженедельному детализированному отчету Wildberries на основе памятки по расчетам.

## Что умеет

- Загружает `.xlsx` файл отчета
- Читает первый лист Excel
- Строит сводную таблицу по ключевым полям
- Считает основные пункты 1–7 по памятке
- Показывает итог прямо в браузере
- Позволяет скачать результат в Excel

## Структура проекта

- `index.html` — основное приложение
- `.gitignore` — исключения для Git
- `README.md` — краткая инструкция

## Как загрузить на GitHub

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы из этой папки.
3. Убедитесь, что `index.html` лежит в корне репозитория.
4. Сделайте первый commit и push.

## Как задеплоить на Vercel

1. Зайдите в [Vercel](https://vercel.com/).
2. Войдите через GitHub.
3. Нажмите **New Project**.
4. Выберите нужный GitHub-репозиторий.
5. Нажмите **Deploy**.
6. После деплоя получите ссылку вида `https://your-project.vercel.app`.

## Важно

- Это статическое приложение, сервер не нужен.
- Для работы нужен браузер и Excel-файл отчета.
- Если структура WB-отчета изменится, может потребоваться обновить сопоставление колонок в `index.html`.
